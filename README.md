# Tic Tac Toe

Only for Linux/MacOs
===============

## Getting started

### Prerequisites

- Linux/MacOs
- GCC compiler

### 1. Clone the repository
```shell
git clone https://github.com/RickinShah/Tic-Tac-Toe.git
cd Tic-Tac-Toe
```

### 2. Compile the project
```shell
g++ main.cpp -o main
```

### 3. Run the project
```shell
./main
```
